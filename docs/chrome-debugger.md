# Debugging in Google Chrome

## WebUSB

Internal: `chrome://usb-internals`

Logging: `chrome://device-log/?refresh=1&types=USB`

## Web Bluetooth

Internal: `chrome://bluetooth-internals`

Logging: `chrome://device-log/?refresh=1&types=Bluetooth`
